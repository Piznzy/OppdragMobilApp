Velkommen til din Expo-app 👋

Dette er et Expo
-prosjekt opprettet med create-expo-app
.

Kom i gang
Installer avhengigheter
npm install
Start appen
npx expo start

I outputen vil du finne alternativer for å åpne appen i:

utviklingsbuild
Android-emulator
iOS-simulator
Expo Go
, en begrenset sandbox for å prøve apputvikling med Expo

Du kan begynne å utvikle ved å redigere filene inne i app-mappen. Dette prosjektet bruker filbasert routing
.
